# 🌍 OptiCoreMC | Terrain Module v1.0.0 
## This Is Made For 26.1.2

This branch contains the standalone **World Geometry & Logic** optimizations for OptiCoreMC. It is designed specifically for **Purpur 1.20.x** to stabilize chunk generation and eliminate registry errors.

## 🚀 Features
* **Tri-Dimension Stabilization:** Custom references for the Overworld, Nether, and The End.
* **Registry Fix:** Prevents "Density Function" crashes common in custom Purpur builds.
* **Auto-Load Announcement:** Sends a visual confirmation in chat when the pack is loaded or reloaded.
* **Chunk Gen Boost:** Optimized noise-router calls to reduce CPU overhead during exploration.

## 🛠 Installation
1. Download the .zip file containing the datapack.
2. Upload to world/datapacks/

## ✅ Compatibility
* **Server:** PurpurMC (Recommended), PaperMC, or Spigot.
* **Version:** 1.20.4 (Pack Format 26).
* **Tested Engine:** Purpur 26.1.2
* **Requirements:** None. Works independently of the OptiCore Skripts.
